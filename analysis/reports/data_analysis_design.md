# 第3回 開邦高校大同窓会 データ分析設計書

## 1. データ概要

### 1.1 利用可能なデータソース

| データ名 | 内容 | レコード数（推定） |
|---------|------|-----------------|
| 参加者名簿 | Peatixチケット購入・チェックイン情報 | 約500件以上 |
| 事後アンケート | イベント後の満足度・要望調査 | 60件 |

### 1.2 主要な分析用フィールド

**参加者名簿**
- ステータス（チェックイン済み/不参加）
- 何期生か（1期〜37期）
- 在学時の科（理数科/英語科/芸術科/学術探究科）
- お勤め先の業種・名称
- 各種同意事項

**アンケート**
- 5段階評価（料理・ドリンク、会費妥当性、日程、時間帯）
- 情報入手経路
- 開催頻度・時期の希望
- 会費許容額
- 希望プログラム
- 不参加理由（周囲から聞いたもの）
- 協力可能な活動
- 自由回答コメント

## 2. 分析フレームワーク

### 2.1 分析の目的
次回大同窓会の成功に向けて、以下の意思決定を支援する：
1. **開催条件の最適化**（日程、時間、会場、会費）
2. **プログラム企画の改善**
3. **集客・広報戦略の強化**
4. **運営体制の構築**（協力者リスト）

---

## 3. 分析項目と手法

### 3.1 参加率・属性分析

#### 3.1.1 基本統計
```
分析内容:
- 申込者数 vs 実参加者数（チェックイン率）
- 期別参加者数分布
- 学科別参加者数分布
- 業種別参加者数分布

手法: 集計・可視化（棒グラフ、円グラフ）
```

#### 3.1.2 参加率の要因分析
```
分析内容:
- 期別のチェックイン率（どの世代が参加しやすいか）
- 学科別のチェックイン率
- 申込時期と参加率の関係

手法: クロス集計、カイ二乗検定
```

### 3.2 満足度分析

#### 3.2.1 全体満足度
```
分析内容:
- 各評価項目の平均値・中央値・標準偏差
- 評価項目間の相関関係
- 総合的な満足度スコアの算出

手法: 記述統計、相関分析
```

| 評価項目 | 分析の視点 |
|---------|----------|
| 料理・ドリンク | 量・質・提供方法の課題抽出 |
| 会費妥当性 | 価格感度の把握 |
| 開催日程 | 12/28の是非 |
| 開催時間帯 | 15:00-17:30の是非 |

#### 3.2.2 属性別満足度比較
```
分析内容:
- 期別満足度の違い（若い世代 vs 上の世代）
- 学科別満足度の違い

手法: クロス集計、t検定/ANOVA
```

### 3.3 テキストマイニング（自由回答分析）

#### 3.3.1 プログラム・演出へのフィードバック
```
分析内容:
- 頻出キーワード抽出
- ポジティブ/ネガティブ分類
- 具体的な改善要望の整理

期待される発見:
- 音響・視認性の問題
- 交流時間とプログラムのバランス
- 期を超えた交流機会への要望
```

#### 3.3.2 運営・会場へのフィードバック
```
分析内容:
- 会場に関する課題（広さ、動線、設備）
- 飲食に関する課題（量、種類、配置）
- 受付・案内に関する課題

手法: 形態素解析、共起ネットワーク分析
```

### 3.4 次回開催条件の最適化

#### 3.4.1 開催頻度の希望
```
分析内容:
- 希望頻度の分布（毎年/3年/5年/10年）
- 属性別の傾向差

意思決定への示唆: 最適な開催サイクルの決定
```

#### 3.4.2 開催時期の希望
```
分析内容:
- 希望時期の集計（複数選択のため重み付け）
- 曜日の希望（土曜 vs 日曜）
- 帰省タイミングとの関連

意思決定への示唆: 参加率を最大化する日程設定
```

#### 3.4.3 会費設定
```
分析内容:
- 許容金額の分布
- 世代別の価格感度
- 「内容重視」派の割合

意思決定への示唆: 適正会費レンジの設定
```

### 3.5 プログラム企画

#### 3.5.1 希望プログラムの優先順位
```
分析内容:
- 各プログラム案の選択率
- 世代別・学科別の希望傾向
- 組み合わせパターンの分析

主な選択肢:
- 校歌斉唱（芸術科音楽コース有志による合唱つき）
- スマッチョ（青藍寮卒業生によるダンス）
- スマホを活用した参加型企画（クイズ等）
- 恩師によるミニ講話・ミニ授業
- 卒業生有志による音楽・伝統芸能・パフォーマンス
- 在校生の活動紹介
- 当時のスライドショー・思い出ビデオ
- 学科・専門分野ごとの交流コーナー設置
```

#### 3.5.2 今後参加したい活動
```
分析内容:
- 各活動への関心度
- 大同窓会以外のニーズ把握

主な選択肢:
- 全期規模の大同窓会
- 専門分野・職種別交流会
- 県外ネットワーク構築
- オンライン座談会
- テーマ別ワークショップ
- 音楽・芸術イベント
```

### 3.6 広報戦略

#### 3.6.1 情報入手経路分析
```
分析内容:
- 各チャネルの到達率
- 効果的なチャネルの特定
- 世代別の情報接触傾向

主なチャネル:
- 同窓生からの口コミ
- 雄飛会Facebook
- 大同窓会Instagram
- 大同窓会ホームページ
- 公共掲示板のポスター
- 学校からの情報
```

#### 3.6.2 不参加理由の分析
```
分析内容:
- 不参加理由の頻度分析
- 対策可能な理由 vs 不可避な理由の分類

主な理由カテゴリ:
- 仕事の都合（12/29仕事納め等）
- 県外・海外在住で帰省スケジュール不一致
- 開催を知らなかった（広報不足）
- 会費が高いと感じた
- 家庭の事情（子育て・介護等）
- 土曜日開催希望
```

### 3.7 協力者リスト作成

```
分析内容:
- 連絡可能な協力希望者の抽出
- 協力分野別の分類
- 専門スキル・リソースの棚卸し

協力分野:
- 特別授業の講師・アドバイザー
- 次回実行委員（企画・運営）
- 寄付や協賛
- 広報協力（SNS、Web等）
- 雄飛会役員活動
```

## 6. 期待される主要インサイト（仮説）

アンケートの自由回答から、以下の課題が予想される：

### 6.1 会場・設備関連
- **音響問題**: 後方席ではプログラムが聞こえない
- **視認性**: ステージが低く見えない
- **収容力**: 人数に対して会場が狭い
- **座席**: 立食のみで休憩スペース不足

### 6.2 飲食関連
- **量の不足**: すぐになくなる
- **配置**: 料理の場所がわかりにくい
- **種類**: ドリンクの選択肢不足

### 6.3 プログラム関連
- **歓談とのバランス**: 会話に夢中でプログラムを見れない
- **世代間交流**: 期を超えた交流機会が少ない
- **参加型企画**: 一方的なプログラムより参加できるものを希望

### 6.4 日程関連
- **土曜開催希望**: 日曜より土曜を希望する声
- **帰省との調整**: 年末年始は交通費高・スケジュール調整困難

## 付録: サンプルコード（データ読み込み）

```python
import pandas as pd

# 参加者名簿の読み込み
participants = pd.read_csv(
    '【卒業生・教職員用】第3回 開邦高校大同窓会 事後アンケート（回答） - 参加者名簿.tsv',
    sep='\t',
    encoding='utf-8'
)

# アンケート回答の読み込み
survey = pd.read_csv(
    '【卒業生・教職員用】第3回 開邦高校大同窓会 事後アンケート（回答） - フォームの回答 1.tsv',
    sep='\t',
    encoding='utf-8'
)

# 期を数値に変換する関数
def extract_generation(text):
    import re
    match = re.search(r'(\d+)期', str(text))
    return int(match.group(1)) if match else None

survey['期_数値'] = survey['入学期（◯期生）'].apply(extract_generation)
```
